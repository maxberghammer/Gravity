## AI-Assistent Interaktion

Die Antworten des AI-Assistenten sollen in diesem Projekt standardmäßig auf Deutsch erfolgen, außer es wird ausdrücklich eine andere Sprache verlangt.

- Um temporär die Sprache zu ändern, gib dies in der Anfrage an (z. B. "Bitte antworte auf Englisch").
- Technische Begriffe, API-Namen, Klassen- und Methodennamen sowie Visual-Studio __Einstellungen__ und __Befehle__ werden in ihrer Originalsprache belassen.

Diese Richtlinie gilt für alle Teilprojekte
